#### Красно-черное дерево
![[Pasted image 20260320174713.png|607]]

**Скорость операций:**
- Все операции за O(log n)

**Как бинарное дерево, имеет свойство:**
- `ключи всех левых потомков <= k < ключи всех правых потомков`

**Свойства красно-черных деревьев:**
- Каждый корень окрашен либо в красный, либо в черный цвет (значение хранится в отдельном поле - цвете узла)
- Путь через черные узлы всегда короче, чем через красные
- Листья (null-узлы) всегда окрашены в черный цвет
- Верхний (корневой) узел всегда черный.
- Каждый красный узел должен иметь 2 черных дочерних узла
- Путь от узла к его листьям должны иметь одинаковое количество черных узлов

**Вставка и перебалансировка:**
1. Новое значение всегда вставляется в лист (предыдущее null-значение) по формуле `ключи всех левых потомков <= k < ключи всех правых потомков`
2. Новый узел всегда окрашивается в красный цвет. Если эта операция нарушает хотя бы одно из свойств красно-черных деревьев, то узел перекрашивается

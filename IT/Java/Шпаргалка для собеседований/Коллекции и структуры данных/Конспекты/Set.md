#### HashSet
Работает на основе **HashMap**: уникальные значения являются ключами Map, в качестве значения - константа, хранящая new Object()
- Место хранения определяется через `hashCode()` за O(1) ([[Контракт equals() и hashCode()]])
- Вставка и удаление за O(1), при многих коллизиях или плохой hash-функции деградирует до O(n) или O(log n)
- Оптимизация вставки и удаления - реализовать `hashCode()` и `equals()` для равномерного распределения элементов по бакетам
#### LinkedHashSet
Под капотом **LinkedHashMap**. Скорость и сложность аналогичны HashSet
#### TreeSet
Под капотом **TreeMap**. Скорость и сложность на все операции O(log n)

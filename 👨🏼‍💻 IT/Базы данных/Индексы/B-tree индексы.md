Являются наиболее часто встречаемой индексной структурой

Хранят пары «ключ — значение» в отсортированном по ключу виде, что позволяет эффективно выполнять поиск значения по ключу и запросы по диапазонам. Гарантированно сбалансировано, то есть глубина B-дерева с n ключами будет равна O(log n).

Надежность обеспечивается механизмом WAL (Write Ahead Log).

> Быстрые при чтении, но медленные при записи. Индекс на основе B-дерева должен записывать каждый элемент данных по крайней мере дважды: один раз в журнал с упреждающей записью и второй — на саму страницу дерева (и вероятно, снова при разбиении страниц). Записывать приходится сразу всю страницу, даже если в ней поменялось лишь несколько байтов
#### Принцип работы:

- БД разделена на блоки или страницы фиксированного размера, обычно 4 Кбайт (иногда больше), и читают/записывают по одной странице за раз.
- Каждая страница имеет свой адрес/местоположение, благодаря чему страницы могут ссылаться друг на друга аналогично указателям. Из ссылок на страницы строится древо
- Одна из страниц назначается корнем B-дерева, с него начинается любой поиск ключа в индексе. Данная страница содержит несколько ключей и ссылок на дочерние страницы. Каждая из них отвечает за непрерывный диапазон ключей, а ключи, располагающиеся между ссылками, указывают на расположение границ этих диапазонов

![[Pasted image 20260318145223.png]]
Количество ссылок на дочерние страницы на одной странице B-дерева называется **коэффициентом ветвления.** На практике этот коэффициент зависит от дискового пространства, которое требуется для хранения ссылок на страницы и границ диапазонов, но обычно равен нескольким сотням

Для обновления значения необходимо найти по ключу страницу-лист, заменить значение и записать на ключ.

В случае надобности добавить новый ключ следует найти страницу, в чей диапазон попадает новый ключ, и добавить его туда. Если на странице недостаточно места для него, то она разбивается на две полупустые страницы, а родительская страница обновляется, чтобы учесть это разбиение диапазона ключей на части

#### ArrayList
+ хранит любые значения, даже null.
+ удаление из конца O(1), вставка в конец за O(1) или за O(n) при перестройке
+ `+` доступ к элементам по индексу за O(1), по значению за O(n)  
+ `-` вставка / удаление элемента в середину списка за O(n), вызывает перезапись всех элементов
- `-` поиск за O(n)
- `-` вставка в начало и удаление из начала за O(n) из-за перестройки;
#### LinkedList
- двусвязный список, каждый элемент содержит ссылки на предыдущий и следующий
- Не требует сдвига элементов при изменениях
- `+` быстрая вставка и удаление в начале и конце — O(1)
- `-` Медленный доступ по индексу — O(n)
- `-` Повышенные расходы памяти (хранение ссылок)
- `-` Вставка и удаление в середину за O(n)

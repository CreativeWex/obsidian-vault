Обучение модели - это процесс, в рамках которого в несколько итераций происходит настройка функции модели для минимизации ошибок на данных
## Основные этапы
1. [[Подготовка данных]]
2. **Выбор архитектуры**: Определение слоёв, нейронов, активаций (ReLU, sigmoid) и оптимизатора (Adam, SGD).
### Основной итерационный цикл
- **Обучение (тренировка)**: Прямой проход (forward pass) — предсказание; вычисление функции потерь (loss, напр. MSE); обратный проход (backpropagation) — градиенты; обновление весов.
- **Валидация и мониторинг**: Проверка на валидационной выборке, контроль переобучения (early stopping, dropout).
### Оценка результата
- **Тестирование**: Финальная оценка на тестовых данных, метрики (accuracy, F1, ROC-AUC).
## Итерация цикла
Процесс повторяется по **эпохам** (проходам по всему датасету), пока loss не стабилизируется или не достигнут лимит эпох

Лимит эпох задается как число итераций - максимальное количество проходов по датасету.

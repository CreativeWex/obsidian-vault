### 1. Подготовка данных. Создание обучающей выборки
В данном примере обучающая выборка - косинусоида. [[Подготовка данных]]
```python
x_train = torch.rand(100) # 100 случайных чисел от 0 до 1  
x_train = x_train * 20.0 - 10.0 # растягиваем значения от -10 до 10  
  
y_train = torch.cos(x_train)  
  
plt.plot(x_train.numpy(), y_train.numpy(), 'o')  
plt.title('$y = cos(x)$');
```

- `torch.rand()` генерирует набор тензоров ([[Тензоры]]).
- `x_train.numpy()` используется для конвертации тензоров PyTorch в массив NumPy. (Matploblib умеет строить графики только из данных NumPy)

Часто данные из обучающей выборки имеют погрешности, **добавляем погрешность для обучающей выборки**, для этого:
1) Генерируем шум
```python
noise = torch.randn(y_train.shape) / 5.  
  
plt.plot(x_train.numpy(), noise.numpy(), 'o')  
plt.axis([-10, 10, -1, 1]) # масштабирование
plt.title('Гауссовский шум');
```
2) Добавляем шум к предыдущим данным чтобы получить данные с погрешностями
```python
y_train = y_train + noise  
plt.plot(x_train.numpy(), y_train.numpy(), 'o')  
plt.title('Зашумленный cos(x)')  
plt.xlabel('x_train')  
plt.ylabel('y_train')
```
3) Изменяем размерность значений
```python
x_train.unsqueeze_(1)  
y_train.unsqueeze_(1);
```
Модель работает с другой размерностью, приводим значение тензоров
```
До unsqueeze
tensor([1., 2., 3.])
После
tensor([[1.],[2.],[3.]])
```
